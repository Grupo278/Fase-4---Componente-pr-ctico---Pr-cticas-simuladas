Sistema Integral de Gestión  de Clientes, Servicios y 
Reservas
Descripción
Sistema orientado a objetos desarrollado en Python para gestionar clientes, servicios y reservas.

Funcionalidades
- Registro de clientes
- Gestión de servicios
- Reservas
- Manejo de excepciones
- Archivo de logs

Tecnologías
- Python
- GitHub
- Programación Orientada a Objetos
